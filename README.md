# Vite + Tailwind + Vanilla JS (AK47)

Starter project scaffolded in place.

Quick start

```bash
cd /home/smartwade/Desktop/\
  Scrimba\ Projects/2025/November/AK47
npm install
npm run dev
```

This will start Vite dev server. Open the printed local URL (normally http://localhost:5173).

Files
- `index.html` — app entry
- `src/main.js` — vanilla JS example
- `src/styles.css` — Tailwind entry (contains @tailwind directives)
- `tailwind.config.cjs`, `postcss.config.cjs` — Tailwind/PostCSS config

Next steps
- Run `npm run build` to build production assets.
- Deploy `dist/` to a static host.
